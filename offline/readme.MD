#浏览器与服务器交互过程
###资源文件没有被改变
```
1.URL输入页面,回车
2.浏览器发现有本地缓存html,使用本地缓存的html页面
3.浏览器解析html,发现有本地缓存中的资源文件
4.浏览器像服务器请求manifest文件
5.服务器返回一个304,通知manifest没有变化
6.浏览器从本地读取资源文件
```
###资源文件被改变
```
1.URL输入页面,回车
2.浏览器发现有本地缓存html,使用本地缓存的html页面
3.浏览器解析html,发现有本地缓存中的资源文件
4.浏览器像服务器请求manifest文件
5.服务器返回更新后的manifest
6.浏览器请求所有文件,包括html文件
7.服务器返回本地缓存的资源文件
8.浏览器对本地文件进行更新,并且存入所有新的资源文件,并且触发一个事件,通知本地缓存被更新.
```
